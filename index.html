Introducción

org-roam es una extensión para Emacs que permite la gestión de notas vinculadas de manera similar a Roam Research.

Este documento técnico cubre los aspectos básicos y avanzados de org-roam.

Para comenzar, es importante entender qué es org-roam y cómo puede mejorar tu flujo de trabajo en Emacs.

org-roam facilita la creación de un sistema de conocimiento interconectado utilizando org-mode.

Con org-roam, puedes vincular notas fácilmente y navegar entre ellas.

Beneficios
  • Organización Interconectada: org-roam facilita la organización de notas de manera interconectada, permitiendo un flujo de trabajo más natural y eficiente. Las notas pueden enlazarse entre sí, lo que ayuda a mantener un sistema de conocimiento cohesivo y bien estructurado
  • Creación y Navegación de Notas: Permite la creación y navegación de notas de forma rápida y sencilla. Con comandos intuitivos, los usuarios pueden crear nuevas notas, enlazarlas a otras y navegar entre ellas sin esfuerzo, mejorando la productividad.
  • Gráfico Visual de Conexiones: Proporciona un gráfico visual que muestra las conexiones entre las notas. Esto ayuda a los usuarios a identificar relaciones y patrones en su información, facilitando una comprensión más profunda de sus datos.
  • Herramientas Avanzadas: org-roam ofrece herramientas avanzadas como la captura de notas diarias y la gestión de alias y referencias. Estas características adicionales permiten a los usuarios mantener sus notas organizadas y accesibles, mejorando aún más su flujo de trabajo.
  • Integración con org-mode: Se integra perfectamente con org-mode, aprovechando todas las características y beneficios de este potente modo de Emacs. Esto incluye el soporte para todas las funcionalidades de org-mode, como el manejo de tareas, la planificación y la estructuración de documentos, lo que convierte a org-roam en una herramienta extremadamente versátil para la gestión de conocimiento
Instalación

Para instalar org-roam, necesitas tener Emacs y org-mode configurados previamente.

Agrega las siguientes líneas a tu archivo de configuración de Emacs:

(use-package org-roam
  :ensure t
  :init
  (setq org-roam-v2-ack t)
  :custom
  (org-roam-directory "~/org-roam/")
  :config
  (org-roam-db-autosync-mode))

Después de agregar las líneas anteriores, reinicia Emacs y ejecuta M-x org-roam-db-build-cache para construir la base de datos de org-roam.

Configuración

La configuración de org-roam se puede personalizar para adaptarse a tus necesidades.

Puedes cambiar el directorio donde se almacenan las notas de org-roam modificando la variable org-roam-directory.

Por ejemplo, para cambiar el directorio a ~/mis-notas/, usa el siguiente código:

(setq org-roam-directory "~/mis-notas/")

Además, puedes configurar las plantillas de captura de org-roam para personalizar la creación de nuevas notas.

Uso básico

Para crear una nueva nota en org-roam, utiliza el comando M-x org-roam-node-find.

Para vincular notas, simplemente escribe [[ y selecciona la nota a la que quieres vincular.

org-roam también permite buscar y filtrar notas utilizando el comando M-x org-roam-node-find.

Para ver el gráfico de notas, utiliza el comando M-x org-roam-graph.

El gráfico de org-roam te permite visualizar cómo están interconectadas tus notas.

Comandos avanzados

org-roam ofrece varios comandos avanzados para mejorar tu experiencia.

Por ejemplo, el comando M-x org-roam-dailies-capture-today te permite capturar notas diarias.

El comando M-x org-roam-alias-add te permite agregar alias a una nota existente.

  • Capturar notas diarias.
  • Agregar alias a notas.
  • Agregar referencias a notas.
  • Sincronizar la base de datos.

Puedes utilizar M-x org-roam-ref-add para agregar referencias a una nota.

Para sincronizar automáticamente la base de datos de org-roam, asegúrate de tener activado org-roam-db-autosync-mode.
